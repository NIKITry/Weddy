# Фоновые изображения

## Добавление фонового изображения

1. Сохраните ваше изображение в эту папку с именем `wedding-background.jpg`
2. Рекомендуемые параметры:
   - Формат: JPG или PNG
   - Размер: минимум 1920x1080px (для лучшего качества)
   - Оптимизация: сжатие для веб (чтобы не замедлять загрузку)

## Текущее изображение

Файл `wedding-background.jpg` будет использоваться как анимированный фон на странице приглашения.

Фон будет:
- Плавно двигаться (анимация slowMove)
- Иметь прозрачность 15% (opacity: 0.15)
- Масштабироваться для создания эффекта движения

## Альтернативные имена файлов

Если вы хотите использовать другое имя файла, измените путь в `InvitationPage.cs`:
```csharp
background-image: url('/images/backgrounds/ваше-имя-файла.jpg');
```
